Mini ChatGPT - Local HTML Chatbot using OpenAI API

INSTRUCTIONS:

1. Replace the placeholder 'YOUR_API_KEY_HERE' in index.html with your actual OpenAI API key.

2. In a terminal, run the server with:
   python3 server.py

3. Open your browser and go to:
   http://localhost:8000

That's it!

Note: For your security, DO NOT share your OpenAI API key or upload this to public servers.
